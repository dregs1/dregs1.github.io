Instruções para a adição deste repo:


Ir para o Kodi gestor de ficheiros.
Clicar em "Adicionar fonte"
O endereço para a fonte é https://dregs1.github.io (Dar o nome de "Dregs Repo").
Ir para "Addons"
Em Addons, instalar de um ficheiro zip. Quando perguntar pela localização, selecionar "Dregs Repo", e instalar repository.dregs-2.36.zip.
Voltar para Addons,mas desta vez, selecionar "Instalar do repositório"
Selecionar o "Dregs Repository"
!!!Agora o resto fica á vossa escolha!!!
